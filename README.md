# nomdev #

A cross-platform front-end for generating and building C++ projects.

## Projects Using nomdev ##

* [i8degrees/nomlib](https://github.com/i8degrees/nomlib)
* [i8degrees/ttcards](https://github.com/i8degrees/ttcards)

## Dependencies ##

* [Ruby 1.9.3-p448](http://rubyinstaller.org/downloads/)
* [cmake v2.6+](http://www.cmake.org/)
* [cloc](http://cloc.sf.net/)
